ATS MAP EXTRACTOR TOOL

IMPORTANT:
TruckLib 0.4.0 only supports .NET 10.
This project must target net10.0 and requires the .NET 10 SDK installed.

Run:
dotnet --list-sdks

If you do not see a 10.x SDK, install .NET 10 SDK first.

Then run:
dotnet run --project OverWatchELD.Tools.AtsMapExtract

Output files:
LiveMap/data/ats_pois.json
LiveMap/data/ats_roads.json
